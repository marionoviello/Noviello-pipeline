# Mapa Scavone 22ª ed. (2026) → Ecossistema Noviello

> **Obra de referência:** Manual Prático de Direito Imobiliário, Luiz Antonio Scavone Junior, GrupoGen — 22ª edição (2026), ISBN 9788530998462. 2.304 páginas, 8 Livros, 43 capítulos, ~75 modelos editoriais.
>
> **Função deste arquivo:** referência cruzada estrutural. Permite ao ecossistema Noviello (i) localizar conteúdo doutrinário no livro para aprofundamento; (ii) saber em qual skill cada modelo editorial foi distribuído; (iii) identificar capítulos cuja fundamentação substantiva precisa atualização.
>
> **Tratamento operacional:** os modelos do livro são **estrutura processual aproveitável + fundamentação a verificar**. A obra é cuidadosamente atualizada na 22ª edição, mas qualquer redação Noviello passa pela verificação cruzada com `references/atualizacoes-legislativas-2022-2026.md` (especialmente Lei 14.382/2022, Lei 14.711/2023, Provimento 149/2023 CNJ, EC 132/2023, LC 214/2025 e LC 227/2026).
>
> **Última atualização do mapa:** 2026-05-04.

## 6. Como o ecossistema usa este mapa

1. **Skill substantiva busca aprofundamento doutrinário** → consulta este arquivo, localiza capítulo relevante no Scavone, cita a página para o cliente ou para si mesma.
2. **Vault recebe peça nova** que talvez tenha casa melhor → consulta este mapa, identifica skill primária preferencial.
3. **Master diagnostica caso complexo** → este mapa indica o capítulo do Scavone que oferece visão integrada do tema, antes de delegar à sub-skill.
4. **Radar legislativo identifica nova lei** → este mapa indica quais capítulos do Scavone ficam afetados, quais skills herdam o impacto, e quais modelos precisam revisão substantiva.

**Forma de citação interna:** "Scavone 22ª ed., Livro X, Cap. Y, p. NN — [tema]". Sem reprodução de conteúdo doutrinário; este mapa é referência cruzada, não substitui consulta direta à obra.
