# Formatos de Conteúdo — Especificações Detalhadas

## Artigo Longo de Blog (Segunda-feira)

**Extensão:** 1.200–2.000 palavras  
**Público:** Adultos entre 45–75 anos, familiares de idosos, herdeiros

### Estrutura obrigatória:

```
[H1] Título com palavra-chave principal (55–65 caracteres)
[Meta-description] ≤ 155 caracteres | inclui palavra-chave + benefício

[Introdução] 120–180 palavras
- Problema ou situação-gatilho do leitor
- O que o artigo resolve
- Frase de autoridade do Dr. Mario Noviello

[Índice clicável] — links âncora para cada seção H2

[Seções H2] — mínimo 4 seções
- Cada seção: 200–400 palavras
- Box de destaque: use para conceitos jurídicos importantes
  Formato: ⚖️ **Saiba que:** [definição em linguagem acessível]
- Exemplos práticos em cada seção

[FAQ] — mínimo 3 perguntas
- Usar schema markup FAQ (para SEO)
- Perguntas no formato que o leitor perguntaria ao Google
- Respostas: 80–120 palavras cada

[CTA Final]
"Está passando por essa situação? Fale agora com o Dr. Mario Noviello.
→ Consulta pelo WhatsApp: [link]
→ Ou acesse: noviello.adv.br"
```

**Checklist SEO:**
- [ ] Palavra-chave no H1, primeiro parágrafo, e pelo menos 2 H2s
- [ ] Alt text em todas as imagens
- [ ] Link interno para ao menos 1 outro artigo do blog
- [ ] Link externo para legislação (leijur.com.br ou planalto.gov.br)
- [ ] Slug da URL: curto, com palavra-chave, sem acentos

```

[Pergunta 3 como H2] — ...
[Pergunta 4 como H2] — ...
[Pergunta 5 como H2] — ...

[Conclusão + CTA] 80–100 palavras
```

## Carrossel Instagram

**Slides:** 8–12  
**Dimensão:** 1080×1080 px (quadrado) ou 1080×1350 px (retrato)

### Modelo de estrutura:

| Slide | Conteúdo | Layout |
|---|---|---|
| 1 | Título-gancho (máx. 6 palavras) + subtítulo (máx. 12 palavras) | Fundo claret, texto branco/creme, Cinzel |
| 2 | "O que você vai aprender:" — lista de 3–4 bullets | Fundo creme, texto claret |
| 3–9 | 1 conceito por slide: título H3 curto + texto (máx. 40 palavras) + ícone | Alternado: creme / claret |
| 10 | Box de alerta ou dado impactante (estatística ou lei) | Fundo prata/cinza claro |
| 11 | Resumo: "Lembre-se:" + 3 pontos-chave | Fundo creme |
| Final | CTA: "Salve e compartilhe" + @novielloadv + WhatsApp + OAB | Fundo claret, texto branco |

### Tipos de carrossel recorrentes:

- **Mito vs. Verdade:** Slide 1 = mito em vermelho / Slide 2 = verdade em verde → repita para 4–5 mitos
- **Passo a passo:** Cada slide = 1 etapa numerada
- **5 Riscos / Erros / Direitos:** 1 risco por slide com ícone de alerta
- **Checklist:** Cada slide = 1 item de checklist com checkbox visual

**Caption do carrossel:**
```
[Frase de impacto — primeira linha visível]

[2–3 parágrafos curtos explicando o tema]

[Pergunta de engajamento para comentários]

💬 Tem dúvida sobre [tema]? Manda aqui nos comentários.
📲 @novielloadv | Consulta: link na bio

#hashtag1 #hashtag2 #hashtag3
```

## Roteiro de Reels / TikTok

**Duração Reels:** 30–60 segundos  
**Duração TikTok:** 45–90 segundos

### Estrutura de roteiro:

```
=== HOOK (0–3 seg) ===
[Texto em tela + fala] — pergunta ou afirmação de impacto
Exemplo: "Você sabia que um inventário atrasado pode te custar 40% do patrimônio?"

=== DESENVOLVIMENTO (3–40 seg) ===
Ponto 1: [frase curta + texto em tela 3–5 palavras]
Ponto 2: [frase curta + texto em tela 3–5 palavras]
Ponto 3: [frase curta + texto em tela 3–5 palavras]
(máx. 4 pontos para vídeos curtos)

=== SOLUÇÃO / AUTORIDADE (40–55 seg) ===
"A solução é [X]. E é mais simples do que você imagina."
Posicionar o escritório como especialista sem prometer resultado.

=== CTA VERBAL + VISUAL (55–60 seg) ===
Fala: "Siga @novielloadv para mais conteúdo sobre [nicho]"
Texto em tela: "Consulta gratuita → link na bio"
Frame final: logo + OAB
```

### Séries temáticas para TikTok:

- **"Direito do Idoso Ep. X"** — direitos pouco conhecidos
- **"Planejando sua herança Ep. X"** — passo a passo do planejamento sucessório
- **"Imóvel Irregular Ep. X"** — como regularizar cada tipo de situação
- **"Plano de Saúde Negou? Ep. X"** — direitos e como acionar

## Copy de Anúncio (Meta Ads)

### Estrutura:

```
HEADLINE (≤ 40 caracteres):
[Benefício direto ou pergunta]

TEXTO PRIMÁRIO (125 chars para preview / até 500 total):
[Linha 1: dor ou situação do público]
[Linha 2: o que o escritório faz a respeito]
[Linha 3: prova social ou dado de credibilidade]
[Linha 4: CTA + link]

DESCRIÇÃO DO LINK (≤ 30 caracteres):
[Complemento da headline]

CTA DO BOTÃO: "Fale Conosco" ou "Saiba Mais"
```

### Públicos recomendados por tema:
- Geriátrico: 50–75 anos, interesses em saúde do idoso, INSS, família
- Sucessório: 45–65 anos, proprietários de imóveis, empresários
- Imobiliário: 30–60 anos, interessados em imóveis, reforma tributária
- Previdenciário: 55–70 anos, aposentadoria, benefícios INSS
